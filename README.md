
# 项目概况
- 搭建一个基本的基于springMvc的web开发脚手架，方便以后快速开发。
- 实现：
- 1. 前端请求映射处理；
- 2. dao层使用hiberante做持久化处理；
- 3. 数据库连接池使用alibaba druid；
- 4. 日志使用slf4j api，实现使用log4j，输入控制台，文件等；
- 5. 返回httpMessge支持json转化；

- 环境
- 1. jdk 1.8
- 2. maven 3.3.3
- 3. tomcat 9
- 4. idea 2017.2.5 / Eclipse Neon
